## SMSwithoutBorders OAuth-2.0-authentications/ Token/ User management-API

### Installation

* Install all node packages
```
npm install
```

### Setup
* Create configuration file
```
touch config.json
```
* Setup configuration file
```
{
    "PORT": "",
    "HOST": "",
    "OWNER": "",
    "DATABASE": "",
    "PASSWORD": "",
    "SESSION_SECRET": ""
}
```
### Start Server
* With NPM
```
npm start
```
* With Node
```
node server.js
```
